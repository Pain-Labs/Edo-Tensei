Edo Tensei

When your AI quota runs out mid-task, switching IDEs shouldn't mean losing your context. One click to hand off to the next agent.

⬇ Install Extension View on GitHub

🔍 Multi-IDE Extraction

Automatically scans local session files from 7+ AI coding assistants — Copilot, Cursor, Claude Code, Windsurf, Kiro, Codex, and Antigravity — all from one sidebar.

⚡ Token-Efficient Handoff

Path mode (default) outputs the session file path + a per-IDE reading guide. The receiving agent reads only what it needs, keeping your new session lean.

📋 One-Click Resurrect

Click any session to instantly copy a ready-to-paste handoff prompt. Right-click for more options like Export, Preview, or path-only Reference prompts.

🗂️ Project-Scoped View

Scan only sessions matching your current workspace, or pull every historical session across all IDEs. Three-level grouping: IDE → Project → Session.

📜 Agent Skill Generator

Generate structured edo-tensei skill/rule files (SOPs) for Claude Code, GitHub Copilot, Kiro, Antigravity, Cline, Gemini CLI, and Cursor.

💾 Export to .edo_tensei/

Save handoff prompts as Markdown files, organized by IDE and project. Automatically prompts to add the folder to .gitignore.

🔒 Local-First Privacy

Everything runs on your machine. No data is sent to any external server. Reads only the local files your installed IDEs already write.

⚠️ Known Limitations

  • macOS / Linux: Not yet supported — Windows only.
  • Trae: Not yet supported. Local databases use SQLCipher encryption with no public key.
  • Windsurf: Currently disabled. Session files use a binary protobuf format; reliable local parsing is pending.
  • Antigravity: Reads from overview.txt (a preview log). Messages are truncated at ~900 characters per turn. The complete conversation history is stored in Antigravity's cloud and is not accessible locally.

Installation

Open VS Code Quick Open (Ctrl+P) and paste:

ext install Pain-Labs.edo-tensei
